## Installation

<InstallationTabs item="signature">
```tsx file=../../registry/spell-ui/signature.tsx
```
</InstallationTabs>

> Place the [LaStoriaBoldRegular.otf](https://spell.sh/LaStoriaBoldRegular.otf) font file inside your `public` directory or update the path in the component to load a different font.
