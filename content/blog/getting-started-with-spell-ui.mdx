## Introduction

[Spell UI](https://spell.sh) is a collection of animated UI components for [React](https://react.dev) and [Tailwind CSS](https://tailwindcss.com). This guide covers setup and first use.

## Installation

Install components directly using the [shadcn/ui](https://ui.shadcn.com) CLI:

```bash
npx shadcn@latest add "https://spell.sh/docs/tilt-card.md"
```

## What Makes Spell UI Different

Spell UI ships interactive components with smooth animations powered by [Motion](https://motion.dev). Each component is:

- **Copy-paste friendly** - No heavy dependencies, just copy the code
- **Customizable** - Built with Tailwind CSS for easy theming
- **Accessible** - Follows web accessibility standards
- **Performant** - Targets smooth 60fps animations
